# Delivery Route Optimizer

Sistema de otimização de rotas para entregas utilizando grafos e o algoritmo de Dijkstra.

## 📌 Sobre o projeto
Este projeto simula um sistema de delivery que calcula a melhor rota entre um restaurante e um cliente, passando por diferentes pontos da cidade.

## 🧠 Tecnologias
- Python
- Estrutura de Dados (Grafos)
- Algoritmo de Dijkstra

## ⚙️ Como funciona
O sistema representa ruas como nós de um grafo e calcula o caminho mais curto com base no custo (distância).
